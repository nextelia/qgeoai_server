# YOLO Models Directory

Place pretrained YOLO models here.

## Required Models

Download from: https://docs.ultralytics.com/models/yolo11/

### Detection
- `yolo11n.pt` (6.5 MB) - Nano
- `yolo11s.pt` (9.7 MB) - Small
- `yolo11m.pt` (20.1 MB) - Medium
- `yolo11l.pt` (25.3 MB) - Large
- `yolo11x.pt` (56.9 MB) - Extra Large

### Segmentation
- `yolo11n-seg.pt` (6.9 MB)
- `yolo11s-seg.pt` (10.1 MB)
- `yolo11m-seg.pt` (20.6 MB)
- `yolo11l-seg.pt` (26.1 MB)
- `yolo11x-seg.pt` (58.8 MB)

### OBB (Oriented Bounding Boxes)
- `yolo11n-obb.pt` (6.9 MB)
- `yolo11s-obb.pt` (10.1 MB)
- `yolo11m-obb.pt` (20.6 MB)
- `yolo11l-obb.pt` (26.1 MB)
- `yolo11x-obb.pt` (58.8 MB)

## Usage

1. Download the model(s) you need
2. Place them in this `models/` folder
3. QModel Trainer will use them automatically
